# Statistical Modeling 2026 Carbon Peak Prediction

2026 年全国大学生统计建模大赛作品：**长三角地级市碳排放时空演化特征、影响因素分析及碳达峰预测——基于面板回归与 LSTM 模型**。

本项目以长三角 41 个地级市 2003-2022 年面板数据为基础，分析区域碳排放的时空演化特征、核心影响因素，并使用 LSTM 模型开展多情景碳达峰预测。

## Project Highlights

- **Spatiotemporal Analysis:** 使用空间自相关、Moran's I、LISA 等方法刻画碳排放空间集聚特征。
- **Panel Regression:** 构建双向固定效应面板模型，分析经济发展、产业结构、人口、电力消费等因素对碳排放的影响。
- **LSTM Prediction:** 基于深度学习模型进行多情景碳达峰预测，比较基准、低碳转型和强化减排情景。
- **Policy Analysis:** 基于实证结果提出长三角区域协同减排和城市差异化达峰路径。

## How to Run

### MATLAB workflow

Recommended environment:

- MATLAB R2019b or later
- Deep Learning Toolbox for the LSTM module

Run from the `code/MATLAB/` directory:

```matlab
run('main.m')
```

The original competition workflow expected all raw Excel/CSV inputs to be available in the MATLAB working directory. This public repository keeps the cleaned panel dataset and generated outputs; raw reference materials and personal submission files are excluded.

### Python scripts

Recommended Python packages:

```text
numpy
pandas
scipy
torch
```

Example:

```bash
python code/python/lstm_predict.py
```

Some scripts may expect intermediate `.mat` or raw data files generated by the MATLAB pipeline.
